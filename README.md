# yingy-first-project

## 项目介绍
这是yingy的第一个GitHub测试项目，用于学习GitHub远程仓库的创建、README.md文件的规范编写，以及Git版本控制的基础操作，是个人Git入门学习的实践项目，用于熟悉GitHub的基本使用流程。

## 使用方法
1.  克隆仓库到本地：执行 `git clone https://github.com/gaoyangyin/yingy-first-project.git` 命令，将仓库拉取到本地
2.  本地修改后提交：在本地修改文件后，通过 `git add`、`git commit` 命令提交修改，再通过 `git push` 推送到远程仓库
